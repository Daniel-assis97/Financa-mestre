# 💵 Finança Mestre

Dashboard financeiro interativo para controle de entradas e saídas mensais. Este projeto foi desenvolvido para facilitar a organização financeira pessoal com uma interface moderna, responsiva e persistência de dados local.

## 🚀 Funcionalidades

- **Controle de Fluxo:** Registro de entradas e saídas com descrição e valor.
- **Cálculo Automático:** Visualização em tempo real do total de entradas, saídas e saldo final.
- **Persistência de Dados:** Os dados são salvos no navegador (LocalStorage), garantindo que as informações não sejam perdidas ao atualizar a página.
- **Design Responsivo:** Layout adaptado para dispositivos móveis, garantindo boa usabilidade em qualquer tamanho de tela (Mobile First).
- **Feedback Visual:** Valores negativos são destacados em vermelho para fácil identificação e ícones intuitivos para o tipo de transação.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica e acessível.
- **CSS3:** Estilização avançada com animações (`@keyframes`), gradientes e foco em responsividade extrema.
- **JavaScript (ES6+):** Lógica funcional para manipulação do DOM, cálculos matemáticos e persistência via Web Storage.

## 📦 Como usar o projeto

1. Faça o download dos arquivos ou clone o repositório.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estejam na mesma pasta.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. **Hospedagem:** Para colocar o projeto online, recomenda-se o uso do [Netlify](https://www.netlify.com/) ou [Vercel](https://vercel.com/).

## 📝 Detalhes Técnicos & QA

- **Responsividade:** Tabela configurada com `table-layout: fixed` e `text-overflow: ellipsis` para evitar quebras de layout em telas pequenas.
- **UX:** Uso de `autocomplete="off"` e `focus()` automático nos inputs para agilizar a inserção de dados.
- **Acessibilidade:** Favicon customizado (💵) via SVG para identificação rápida na aba do navegador.
